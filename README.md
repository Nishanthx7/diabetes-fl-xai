# 🧬 Federated Learning-Based Diabetes Risk Prediction (XAI)

## 🚀 Overview

This project implements an **end-to-end machine learning pipeline** for diabetes prediction using:

* Federated Learning (privacy-preserving ML)
* Explainable AI (SHAP)
* Advanced ML models (XGBoost, LightGBM, Random Forest)

---

## ⚡ Features

✔ Data preprocessing with missing value handling
✔ Feature engineering (custom health indicators)
✔ SMOTE for class imbalance handling
✔ Multiple ML models:

* Random Forest
* XGBoost
* LightGBM

✔ Federated Learning simulation (multi-client setup)
✔ SHAP-based explainability
✔ Interactive diabetes risk predictor

---

## 📊 Outputs

* ROC Curve Comparison
* Confusion Matrices
* Model Performance Summary (CSV)
* SHAP Feature Importance Graph
* Federated Learning Accuracy per Round
